IOM Missing Migrants Project

To practice my leaflet and shinyapp skills I started this small project with the IOM Missing Migrants dataset. I optained the data from kaggel.

The dataset encompasses details regarding the fatalities and disappearances of migrants globally. It is meticulously compiled from diverse sources, including IOM field offices, media reports, and surveys. Updated on an annual basis, the dataset spans from 2014 to 2020. Curated by the International Organization for Migration's Missing Migrants Project, this repository serves as a comprehensive database documenting missing and deceased migrants.

I designed a Leaflet map to visually represent the dataset, showcasing the reported incidents. Additionally, the bar plot displays the number of incidents per migration route. Users have the flexibility to choose the year, month, and region of interest, with both the map and bar plot updating dynamically based on their selections.You can access the code for my Shiny app here.
